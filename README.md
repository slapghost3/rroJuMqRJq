# 医疗预约与诊断系统 python636

## 项目概述

本项目是基于 Python 的 Flask 框架开发的医疗预约与诊断系统，它集成了用户管理、预约挂号、在线咨询和病历管理等核心功能，服务于管理员、医生和患者三大用户角色。

## 技术栈

- 后端：Python、Django
- 数据库：MySQL
- 前端：Vue.js

## 功能模块

### 系统角色

- 管理员
- 医生
- 患者（用户）

### 管理员功能

- 用户管理
- 医生管理
- 科室信息管理
- 症状类型管理
- 疾病信息管理
- 就诊信息管理
- 预约挂号管理
- 诊断病历管理
- 评价信息管理
- 预约取消管理
- 在线咨询管理
- 充值信息管理
- 系统管理
- 我的信息

### 医生功能

- 就诊信息管理
- 预约挂号管理
- 诊断病历管理
- 评价信息管理
- 预约取消管理
- 在线咨询管理
- 我的信息

## 系统运行环境

- Python 3.x
- Django 2.x
- MySQL 5.x
- Vue.js 2.x
